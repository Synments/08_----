## 作業環境說明 (Operating Environment) 

本專案採用專業的本地開發工作流，透過高度可擴充的集成開發環境 (IDE) 進行模型開發與測試，確保開發環境的穩定性與隱私性。
- **作業系統**：Windows 11 
- **開發工具**：Visual Studio Code (VS Code)
	- **本地運算**：直接呼叫本地硬體資源，避免了雲端環境的連線延遲與斷線風險。
- **程式語言**：Python 3.12 
- **虛擬環境管理**：透過 `venv` 建立獨立的虛擬環境，嚴格控管套件版本，確保專案的 **可重複性 (Reproducibility)** 與環境隔離。
- **核心套件**：
	- `Pandas` / `NumPy`：負責大規模結構化資料的處理與矩陣運算。
	- `Scikit-learn`：提供隨機森林模型、TF-IDF 向量化及模型評估指標。
	- `NLTK`：用於文本預處理（如：移除停用詞、詞幹提取）。
	- `Re (Regex)`：執行正規表示法清洗，移除 HTML 標籤與特殊符號。
	- `Streamlit`：建立互動式 Web 預測介面。
	- `Joblib`：執行模型持久化 (Serialization)，將訓練好的模型存檔。
